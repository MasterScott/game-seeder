It’s WiiU game, why it’s on PC now?

Well, it’s simple. There is a WiiU emulator, called Cemu, which makes games designed for WiiU to be playable on Windows platform.

Cemu is constantly improving, cause it’s supported by backers on Patreon platform, so anyone can expect a more powerful, more compatible and much faster WiiU emulation in the near future.

But even now, with Cemu v1.7.4d you can play Bayonetta 2 game on your PC, if it’s not an abacus. 4-cores CPU + good GPU are highly recommended.

Just download this repack, install as any other PC game and run it from desktop icon, created by the installer.

New Cemu versions are released on Cemu official site: http://cemu.info. If you become a patron on Patreon – you will receive access to early betas before public release.

The current version of Cemu allows to play Bayonetta 2, but there bugs (like flickering and audio bugs) and perfomance is not the best side.

With next Cemu updates just unzip new version to “cemu” folder, saving keys.txt and settings.bin files just in case.

Shader cache included in this repack for improved game perfomance.
