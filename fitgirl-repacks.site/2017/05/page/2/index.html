Save Fix

The game creates folder & save game files inside cpy_default folder (in documents) you can see the files, but the game don’t recognize them (you can’t CONTINUE or LOAD GAME they’re both greyed out, even if you start a new game) + options aren’t saved + the game introduces the DLCs each time you start it.

Fix #1: Check “Allow files in this folder to have contents indexed” of “C:\Users\YOUR_USERNAME\Documents\CPY_SAVES” folder in folder advanced properties
Fix #2: UN-check “Compress contents to save space” of “C:\Users\YOUR_USERNAME\Documents\CPY_SAVES” folder in folder advanced properties, AND make sure to apply to all folders, sub-folders, and files.
Fix #3: In “CPY.ini” you change the savepath to the partition the game is installed from default. It would become something like SavePath=D:\, after that you can find the saves D:\CPY_SAVES\ and the game can work properly. You can even put saves from Voksi fix, they work flawlessly. Thanks to Hernest.
