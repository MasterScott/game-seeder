Changing the game language

If you install more than one language, you should do the following to change the language. Here’s the example for German.

1) Go to “data\em5\local\localization” folder inside game folder
2) Rename “en” folder to something else, like “en_old”
3) Rename “de” folder to “en”
4) Go to “data\em5” folder
5) Rename “language_en” folder to “language_en_old”
6) Rename “language_de” folder to “language_en”

Now all your campaigns and audio will be in your native language

You can also play Emergency 5 campaigns in Polish and Russian languages, but they are not supported in 2016 and 2017 campaigns at all.
