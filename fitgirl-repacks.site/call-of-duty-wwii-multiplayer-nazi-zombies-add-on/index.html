How to play multiplayer and zombies modes?
  1. Install this addon to the same folder, where singleplayer game is installed
  2. Run it from desktop icon or from “Steam.exe” in game root (it’s not original “Steam.exe”, just the launcher
  3. Select game mode by clicking YES (for MP) or NO (for Zombies)
  4. After alert window, close your Steam application and run it again. Don’t worry, you need to do it once
  5. With Steam running in background, run the game again and you can play MP or Zombies with bots (use Local Game menu). For playing with real players via LAN, read instructions below
