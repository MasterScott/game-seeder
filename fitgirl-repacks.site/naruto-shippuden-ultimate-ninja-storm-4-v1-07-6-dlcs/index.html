Repack Features

  • Based on NARUTO.SHIPPUDEN.Ultimate.Ninja.STORM.4-CODEX ISO release: codex-naruto.shippuden.ultimate.ninja.storm.4.iso (35,576,578,048 bytes)
  • All updates, languages and DLCs installed over, game version is 1.07
  • CODEX crack replaced with 3DM one, but available in NoDVD folder after game installation. Steamworks Fix (by deadmau5+UberPsyX), allowing multiplayer via Steam is also available in NoDVD folder
  • 100% Lossless & MD5 Perfect: all files are identical to originals after installation (when installed in MD5 Perfect Mode)
  • NOTHING ripped, NOTHING re-encoded
  • Selective download feature: you can install any of 4 available voice packs (English, Japanese, Spanish or Portuguese-Brazil) or to install them alltogether in MD5 Perfect Mode
  • When installing a single voice pack, be sure to enable this language later in game options or you’ll hear silence
  • For some reason developers made Portuguese-Brazil and Spanish voiceovers turnable only when OS locale is set to Brazilian or Spanish US
  • Significantly smaller archive size (compressed from 41 to 25.6~27.9 GB, depending on selected languages)
  • Installation takes from 50 minutes (8-cores CPU + SSD) up to 2.5-3 hours (2-cores CPU + HDD)
  • After-install integrity check so you could make sure that everything installed properly
  • HDD space after installation: 41 GB (~56 GB during installation)
  • Use “Language Selector.exe” in game root to change the text/GUI language
  • At least 2 GB of free RAM (inc. virtual) required for installing this repack
