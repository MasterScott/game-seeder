Changing graphics settings

GOG version has a bug when applying GFX settings via “VideoSetup.exe” for non-English languages.
It always writes changes to “C:\Users\Your Username\Documents\my games\The Saboteur™” folder,
even if the language is set to one, which have localized name for “My Games” folder, like “Мои игры” for Russian OS.

To workaround it, set the game language to English in “language_setup.exe”, save settings in “VideoSetup.exe” and then copy
“C:\Users\Your Username\Documents\my games\The Saboteur™\config.ini”to
“C:\Users\Your Username\Documents\YOUR LOCALIZIDED GAMES FOLDER\The Saboteur™\config.ini”, overwriting it.

If you want to play in a window, add “Windowed 1” at the end of the “config.ini”

Setting refresh rate from 60 Hz to anything above will reset the game to default 800×600 resolution.

The player can forge alliances with or against others as well as challenge the computer. Randomly generated maps provide endless variations and can be adapted to the gamer’s needs.

Game Features

  • Grandiose battles of up to 10,000 soldiers on the battlefield
  • 12 playable nations
  • 100 research opportunities
  • 140 different historic buildings
  • Battles ashore and at sea
  • Five singleplayer historical campaigns
  • Infinite variations of battles on randomly generated maps
