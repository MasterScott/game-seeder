Decompression Test

December 4, 2016 FitGirl 370 Comments

Hello, everybody!

I’m planning to implement a new kind of compression algo in my repacks. It gives a slightly better compression ratio + installs faster. But before I implement it in repacks, I have to test the algo on you, my dear users 🙂

So, if you can download ~900 MB from zippyshare, go ahead:

http://www105.zippyshare.com/v/5Bq3xYvE/file.html
http://www105.zippyshare.com/v/nn9Ybg40/file.html
http://www105.zippyshare.com/v/BuGx9EH8/file.html
http://www105.zippyshare.com/v/t3auGm5K/file.html
http://www105.zippyshare.com/v/2gh1bNb5/file.html

UnRAR those files, then run setup-test-1.exe, setup-test-2.exe & setup-test-3.exe one after another.

Please remember/write down installation times for all three installers (each one should be within one minute). Then please report those in comments to this topic and provide your CPU type, RAM amount you have onboard and target drive you installed tests to.

The older your CPU is – the better for test. Thanks in advance!

Sample report:

Test 1: 0:17
Test 2: 0:25
Test 3: 0:14
CPU: Intel i5 6600
RAM: 16 GB
Installed on: HDD

AMD (and old Intel) CPU owners wanted for testing

September 26, 2016 FitGirl 33 Comments

My current DOOM repack cannot be installed on some old CPUs, like AMD Phenoms or Intel Core Duo/Quad. The reason is that the tool used in there uses newer SSE instructions for faster data processing. But since release of my repack new version of the tool became available. And maybe now it supports older CPUs.

So, if you have and old AMD/Intel CPU (will be even better if my DOOM IDDQD repack failed for you), please contact me on one of sites listed. I will give you instructions and test set. Thanks in advance.

Guys, don’t post your configs here – you have to contact me personally to get the test set.
