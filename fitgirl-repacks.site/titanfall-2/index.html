Repack Features

  • Based on Titanfall.2-CODEX ISO release: codex-titanfall.2.iso (45,252,935,680 bytes)
  • Titanfall.2.Update.v2.0.7.0-CODEX (1.6 GB) applied over
  • Missing localizations added (5.2 GB), thanks to Christsnatcher from cs.rin.ru
  • 100% Lossless & MD5 Perfect: all files are identical to originals after installation
  • NOTHING ripped, NOTHING re-encoded
  • Selective download feature: you may skip downloading and installing of multiplayer files (NO, YOU CAN’T PLAY MP IN THIS VERSION!) and language files you don’t need
  • Significantly smaller archive size (compressed from cumulative 48.9 to 16.8~23.2 GB, depending on selected components)
  • Installation takes 35-70 minutes (depending on your system and selected components)
  • After-install integrity check so you could make sure that everything installed properly
  • HDD space after installation: up to 68 GB
  • Use “Language Selector.exe” in game root to change the game language
  • At least 2 GB of free RAM (inc. virtual) required for installing this repack

AMD CPU Setup Fix

If you were unlucky to buy AMD CPU (oh, how do I hate that crap!) for you, get this alternative setup-fitgirl-08.bin and place it over original file in repack folder. Of course, the installation will be slower, than on Intels. But what did you expect from AMD?

Selective Download

You can skip downloading of language packs you don’t need and multiplayer files you can’t use anyway. Here is the list of selective/optional files.

  • setup-fitgirl-optional-multiplayer.bin (download and install only if you need to import this repack to your legit Origin installation)
  • setup-fitgirl-selective-english.bin
  • setup-fitgirl-selective-brazilian.bin
  • setup-fitgirl-selective-chinese.bin
  • setup-fitgirl-selective-french.bin
  • setup-fitgirl-selective-german.bin
  • setup-fitgirl-selective-italian.bin
  • setup-fitgirl-selective-mexican.bin
  • setup-fitgirl-selective-polish.bin
  • setup-fitgirl-selective-russian.bin
  • setup-fitgirl-selective-spanish.bin

In example, if you want to launch the game with German UI/Subtitles/Voiceovers – skip all “selective” files, but download setup-fitgirl-selective-german.bin & all main files (01-08).
