]]> By: Tek http://fitgirl-repacks.site/warning-megaupload-users/#comment-9330 Tue, 20 Sep 2016 14:48:32 +0000 http://fitgirl-repacks.site/?p=1767#comment-9330 “I wonder why the torrent client doesn’t hash the files 100% after re-adding the torrent even if the files are 100% correct.”

It’s because of the way bittorrent chops the files into blocks for transmission: in this case, the last part of one file, and the first of another are split among the same block. If you don’t download one of them and then remove the torrent and add it later to re-check, you’ll be missing the resume data from your torrent client that stored any extraneous blocks that you may have downloaded (of the deselected files in the torrent that shared data with the files you want), and so it will say the file is ‘incomplete’. It’s not though, with the exception of the files getting fucked up by Megaupload recently, which has been fixed.
